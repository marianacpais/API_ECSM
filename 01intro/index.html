Introdução

Contextualização

Este documento fornece as informações necessárias para utilizar a API do Repositório de Dados para Investigação das Equipa Comunitária de Saúde Mental.

Parte da organização e consequente gestão de acessos a este repositório depende da concretização das reformas a implementar futuramente pelo Grupo de apoio técnico à implementação das políticas de saúde (GAPS), bem como da articulação com os Serviços Partilhados do Ministério da Saúde (SPMS) e Equipas Comunitárias de Saúde Mental (ECSM). Por este motivo, é previsível que surjam alterações de fundo à estrutura de dados e consequentemente a esta documentação para melhor acomodar estas mudanças.

Plataforma

É utilizada a plataforma de recolha de dados REDCap, na qual assentam as operações que em seguida se descrevem.

Cada um dos Métodos apresentados na secção “Métodos Suportados” providencia uma ação específica que pode ser desempenhada utilizando esta plataforma.

“API” é o acrónimo de Application Programming Interface (Interface de Programação de Aplicações) e corresponde a uma forma normalizada de interagir com um programa para desempenhar uma determinada tarefa.

Esta API utiliza a norma REST (Representational State Transfer), pelo que a comunição é efetuada por via da web através do protocolo HTTP (Hypertext Transfer Protocol).
