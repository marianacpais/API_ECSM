Para que seja possível executar operações quer de leitura quer de escrita é necessária a obtenção de uma chave de API (*token*), que corresponde a um código com 32 caracteres.

Cada chave de API corresponde a um determinado utilizador e as chamadas à API que esta permite efetuar dependem das permissões atribuidas a cada utilizador.

> NOTA: No caso particular desta implementação, o mesmo utilizador pode necessitar de diferentes chaves de API para executar operações a diferentes conjuntos de dados. Para mais informações, consulte a secção ["Esquema Geral da Estrutura de Dados"](../05esquema-dados/)
